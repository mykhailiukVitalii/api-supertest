# API tests with Jest and SuperTest

## Features

-   TypeScript
-   Jest
-   SuperTest
-   fake-api
-   report

## Requirements

-   node >= 12.x - [how to install Node](https://nodejs.org/en/download/)
-   npm >= 6.14.x - [how to install NPM](https://www.npmjs.com/get-npm)

## Tests Started

Install the dependencies:

```bash
npm install
```

Compile TypeScript:

```bash
npm run build
```

Run api tests:

```bash
npm run test
